# Morning Dashboard — Chrome side panel

A panel that slides out on the right side of Chrome, **next to any website**, with:

- 📝 **Notes** — one running scratchpad. Type anything; it saves automatically. Perfect for keeping notes beside a website you're reading.
- ✅ **To-Do** — add tasks, check them off, see your progress. Unfinished tasks carry over to the next day.
- 📧 **Email** — a placeholder for now; the AI email summary gets wired up in Stage 2 (needs a one-time Gmail setup).

Everything is stored privately in your own browser.

## How to install it (one time, ~2 minutes)

1. Download/copy this `extension` folder onto your computer (remember where you put it).
2. Open Chrome and go to `chrome://extensions` (type it in the address bar).
3. Turn on **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked** (top-left) and select this `extension` folder.
5. "Morning Dashboard" now appears in your extensions. Click the **puzzle-piece icon** in Chrome's toolbar, find Morning Dashboard, and click the **pin** so it stays visible.

## How to use it

- Click the **Morning Dashboard icon** in the toolbar — the panel opens on the right, next to whatever website you're on.
- Switch between **Notes**, **To-Do**, and **Email** using the tabs at the top.
- It stays open as you browse different sites, so your notes are always one click away.
